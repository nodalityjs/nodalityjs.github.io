Grid

The Grid creates an instance of the Simple element, designed to organize child elements in a grid layout.

let elements = [
  {
    type: "grid"
  }
];

In set method of this class, we can use the gap property to set gap within the grid. The react method allows the grid to respond to breakpoints and dynamically adjust its layout.

Arguments:​

| Argument  | Description |
|-----------|-------------|
| `at`      | The breakpoint (in pixels) at which this template becomes active. |
| `template`| An array of strings defining the grid areas for this breakpoint. Each character represents a grid area. |
| `color`   | (Optional) Sets the color of the grid items at this breakpoint. |

In Wrapper class, in this example we can use the mc property sets the height to max-content.

mc: true means sets this css element height to max-content.
