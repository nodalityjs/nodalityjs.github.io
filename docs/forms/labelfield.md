# LabelField

LabelField allows you to create textfield that turns placeholder text into a label.

```js
let elements = [
  {
    type: "labelInput"
  }
];
```
An instance of ```FloatingInput``` class is created:
```js
new FloatingInput()
        .set({
            title: "Your name",
            type: "input"
        })
```
