# FilePicker
Creates a simple filepicker.
```js
let elements = [
  {
    type: "filePicker"
  }
];
```

An instance of ```FilePickera``` class is created.

### Properties
```accept``` - sets the accepted file-type
```js
 FilePickera()
          .set({
              id: "A",
              title: "Add profile picture",
              background: "#3498DB",
              color: "white",
              font: "Arial",
              radius: "3rem",
              accept: "application/pdf"
          });
          // 195644 off the ch. 02/02/25
```
